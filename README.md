# SwappApp - QA Test Deliverables

## 📌 Overview
This repository contains the QA deliverables for the SwappApp test assignment. The assessment focuses on comprehensive manual testing, detailed bug reporting, test case analysis, and API functional testing.

---

## 🔗 Deliverables

- 🐞 **Bug Report (Manual Tests):** [View Document](https://docs.google.com/document/d/10MNcE1pmJB5miLal2kLQ6lhhcZOrFs9S0prVs20kg9g/edit?usp=sharing )
- ✅ **Test Cases & Execution:** [View Spreadsheet](https://docs.google.com/spreadsheets/d/1rjh-1V3BlBEq0zkL-e3qTBYimYggWtNfniPmP86lN0c/edit?usp=sharing )
- 🔌 **API Testing Collection:** Available in the `/api-tests` directory.

---

## 🛠 Tech Stack
- **Manual Testing:** macOS / Chrome
- **API Testing:** Postman
- **Documentation:** Google Workspace / Markdown

---

## ▶️ How to Run API Tests
1. Navigate to the `/api-tests` folder.
2. Import the JSON collection into **Postman**.
3. Ensure the environment variables are set (if applicable).
4. Run the collection via Postman Runner.

## ⚠️ Notes
Due to the assignment's time constraints, the focus was prioritized on **high-impact manual exploration**, **API functional coverage**, and **rigorous bug documentation** rather than full automation scripts.
